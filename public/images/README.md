# Image Requirements for RAC Kitengela Website

## Required Images and Sizes

### Open Graph Images (for social media sharing)
- **og-home.jpg** - 1200x630px - Homepage sharing image
- **og-events.jpg** - 1200x630px - Events page sharing image  
- **og-members.jpg** - 1200x630px - Members page sharing image
- **og-about.jpg** - 1200x630px - About page sharing image
- **og-contact.jpg** - 1200x630px - Contact page sharing image

### Favicon Files
- **favicon.ico** - 32x32px - Browser tab icon
- **apple-touch-icon.png** - 180x180px - iOS home screen icon
- **favicon-32x32.png** - 32x32px - Standard favicon
- **favicon-16x16.png** - 16x16px - Small favicon

### Content Images
- **hero-bg.jpg** - 1920x1080px - Hero section background
- **about-image.jpg** - 800x600px - About section image
- **team-placeholder.jpg** - 400x400px - Default member photo
- **event-placeholder.jpg** - 600x400px - Default event image

## Image Guidelines
- Use high-quality, professional images
- Optimize for web (compress without losing quality)
- Use consistent branding colors (Cranberry #DC143C, Lime #C1FF07)
- Ensure images are accessible with proper alt text
- Consider dark/light theme compatibility

## Tools for Optimization
- TinyPNG for compression
- Canva for creating social media images
- Adobe Photoshop/GIMP for editing
